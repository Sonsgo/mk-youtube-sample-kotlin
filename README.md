# mk-youtube-sample-kotlin


# Kotlin Youtube Sample App
Sample Youtube Application in Kotlin which show how to use Youtube Data API


# How to run this App
- Clone the repo
- Replace in AppConstants with your own Google Client_ID and Youtube_Api_key
- Just run
